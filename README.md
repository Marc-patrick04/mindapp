# 🧠 MindApp - Mental Health & Gambling Risk Detection

## Overview
MindApp is a web application designed to help university students detect early signs of gambling-related mental health risks. Students can take anonymous assessments, receive immediate risk feedback, and optionally login to access peer support groups and professional counseling.

## Features
- ✅ **Anonymous Assessment** - No login required to take the assessment
- ✅ **Multiple Question Types** - Yes/No, Scale (1-5), Checklist, Multiple Choice
- ✅ **Real-time Risk Detection** - Low/Moderate/High risk levels with color coding
- ✅ **Personalized Recommendations** - Based on assessment answers
- ✅ **Support System** - Login to request peer support or counseling
- ✅ **Admin Dashboard** - Manage questions, scoring rules, and support requests
- ✅ **PostgreSQL Database** - Secure data storage

## Tech Stack
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Authentication:** JWT + bcrypt
- **Frontend:** HTML/CSS/JS
